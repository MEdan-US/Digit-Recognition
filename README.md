# Digit Recognition Project in Python

## Project Description
This project implements a digit recognition system using various machine learning techniques in Python. The goal is to accurately identify handwritten digits from images.

## Features
- Utilizes popular libraries such as TensorFlow and Keras.
- Implements Convolutional Neural Networks (CNN) for improved accuracy.
- Provides a user-friendly interface for image input.
- Supports training on the MNIST dataset.

## Installation Instructions
To run this project, you will need to have Python installed on your machine. Follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/MEdan-US/Digit-Recognition.git
   ```
2. Navigate to the project directory:
   ```
   cd Digit-Recognition
   ```
3. Create virtual environment:
    ```
    conda create --name digit_env -y
    conda activate digit_env
    ```
4. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage
To use the digit recognition system:
1. Run the main script:
   ```
   python -m streamlit run app.py
   ```
2. Follow the prompts to input images of handwritten digits.
3. The system will output the predicted digit.
